# odin-recipes
A basic recipe website built as part of the The Odin Project Foundations curriculum.

## Description
A simple multi-page recipe website. This project was built to practice core HTML concepts and Git workflow.

## Skills Demonstrated
- HTML boilerplate structure
- Headings, paragraphs, and lists
- Images with alt text and attribution
- Linking between pages
- Git version control and GitHub Pages deployment
